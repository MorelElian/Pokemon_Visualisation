Generation Impact

Number of Pokemons per generations

Odd generations have more new pokémon compared to even ones. The average number of pokemons per generations is 100, and generation 5 is the most prolific generation

Generation 3 has the easiest pokémon to catch, while generation 4 has the hardest.

What about Secondary Type ?

Secondary types seem to be almost the opposite of primary types: the treemap allows us to visualize this, let's see if we can better see the repartition with a stacked barplot

Pokemon by Primary and second types

We have created a stacked barchart, sorted by Primary Type. Thanks to this graph, we can detect two insights :

First, the Primary Type is directly correlated to the global number of Pokemon

Second, The more a type is a Primary Type, the less is will be a secondary type

Generation and Types

Pokémon Primary Types Prevalence Across generation

A few types don't exist in some generations, like dark, steel and flying in generation 1 and dragon and flying in generation 2.

Flying only exists as primary type in generations 5 and 6.

There is an unusually high amount of poison type pokémon in the first generation, steel type in the third, dark and psychic type in the fifth and fairy in the sixth.
